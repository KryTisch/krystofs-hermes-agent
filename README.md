# Krystofs Obsidian Vault

Dies ist der serverseitige Markdown-/Obsidian-Vault, der per Git mit Mac und Handy synchronisiert werden soll.

Grundregel:
- Obsidian/Git = kuratiertes Langzeitwissen, Quellen, Entscheidungen, Projekt-Wiki.
- Hermes Memory = sehr kurze Fakten, die der Assistant ständig direkt kennen muss.
- Spezialisierte Datenbanken bleiben für operative Daten zuständig.

Startstruktur:
- `00_Inbox/` schnelle Eingänge
- `01_Personal/` persönliche Langzeitnotizen
- `02_Projects/` Projektwissen
- `03_Knowledge/` Themen-/Konzeptwissen
- `04_Sources/` Quellen
- `05_Outputs/` abgeleitete Artefakte
- `_indexes/` Navigation und Übersichten
- `_templates/` Vorlagen
