# Obsidian Sync Entscheidung

Aktueller Stand: Obsidian Sync ist primärer Sync-Weg; Git dient als Backup/Versionierung.

## Begründung

Obsidian Sync ist für Mac, Handy und Headless-Server alltagstauglicher als Git als Primärsync. Git bleibt stark als serverseitige Historie und Rollback-Mechanismus.
